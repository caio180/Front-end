# Sistema Informativo: Fome no Mundo

Este projeto é uma página web informativa sobre a fome mundial.

Ele foi desenvolvido com fins educacionais para praticar conceitos básicos de desenvolvimento web utilizando HTML e CSS, além de conscientizar sobre um dos maiores problemas sociais da atualidade.

## Funcionalidades

- Exibição de informações sobre a fome mundial
- Explicação das principais causas da fome
- Consequências sociais causadas pela insegurança alimentar
- Lista com os fatores que mais contribuem para a fome
- Tabela com dados de pessoas em situação de fome por continente
- Inserção de imagem ilustrativa
- Estilização visual utilizando CSS
- Efeito visual ao passar o mouse sobre a imagem

## Estrutura do Projeto

O projeto foi construído utilizando:

 **HTML5** → responsável pela estrutura e conteúdo da página
 **CSS3** → responsável pela estilização e aparência visual

## Pré-requisitos

- Navegador atualizado (Google Chrome, Edge, Firefox, Opera, etc.)

## Como executar o projeto

### 1 - Clone o repositório

```bash
git clone https://github.com/caio180/Front-end.git
```
### 2 - Acesse a pasta do projeto:

```bash
cd Fome no mundo
```

### 3 - Abra o arquivo HTML:

* Basta abrir o arquivo **index.html** no navegador.


### Objetivo do Projeto:

* O objetivo deste projeto é desenvolver habilidades em estruturação de páginas web e estilização visual, além de apresentar informações importantes sobre a fome mundial e seus impactos sociais.
